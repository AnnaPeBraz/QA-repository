# Authentication - Feature Mapping

## Overview
    The authentication module allows users to:
        - Create an account
        - Sign in 
        - Recover their password
## User Flow
### Signup
    Step 1:
    -> Click Create a new account:
        - First Name 
        - Last Name 
        - Email 
        - Phone (optional) 
    -> Click Next:
         
    Step 2:
        - Select a role 
        - password 
        - confirm password 
        - subscribe to the newsletter (optional) 
        - Data protection policy agreement 
    -> Click Register
    The account is created.

### Login
    -> Enter the email address
    -> Enter the password
    -> Click Login

    The user is redirected to the Dashboard.
    
    -
    Alternative routes:
    - Forgot password
    - Create a new account 

### Forgot password
    -> Click Forgot password 
    -> enter a email 
    -> click change password 
    -> receive the password reset email
    -> Create a new password
    -> Return to the Login page

    -
    alternative routes:
    Login
