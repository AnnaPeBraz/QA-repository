# Authentication - Feature Mapping

## Overview
    The authentication module allows users to:
        -create an account
        -Sign in 
        -recover their password
## User Flow
### Signup
    Step 1:
    -> Click on Create a new account:
        -First Name 
        -Last Name 
        -Email 
        -Phone (optional) 
    -> Click on Next:
         
    Step 2:
        -Choose role 
        -password 
        -confirm password 
        -Newsletter Subscribe (optional) 
        -Data protection policy agreement 
    -> Register

### Login
    email address -> Password -> Login

### Forgot password
    Forgot password -> enter a email -> change password -> receive an email
